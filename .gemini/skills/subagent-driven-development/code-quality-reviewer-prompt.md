# コード品質レビューア プロンプトテンプレート

コード品質レビューア サブエージェントをディスパッチする際にこのテンプレートを使用します。

**目的:** 実装が適切に構築されているか（クリーン、テスト済み、保守可能）を検証します。

**注意:** 仕様準拠レビューが合格した後のみディスパッチしてください。

```
ツール呼び出し (Gemini CLI のコードレビュー機能):
  code-reviewer スキル (または同等の機能) を使用します。

  WHAT_WAS_IMPLEMENTED: [実装者のレポートより]
  PLAN_OR_REQUIREMENTS: [計画ファイル]のタスク N
  BASE_SHA: [タスク実行前のコミットハッシュ]
  HEAD_SHA: [現在のコミットハッシュ]
  DESCRIPTION: [タスクの概要]
```

**コードレビューアの返却内容:** 強み、問題点（致命的/重要/軽微）、評価
