# 知見：using-git-worktrees (Gemini固有)

## AIエージェントへの指示 (Gemini固有)
- **重要:** ワークツリーを作成し、その中で作業を開始した後は、`run_shell_command` や `write_file` などの**ファイルI/Oを伴うすべてのツール呼び出しにおいて、必ず `dir_path` パラメータにワークツリーのフルパスを指定してください。**
- これを怠ると、コマンドはデフォルトのプロジェクトルート（メインワークスペース）で実行され、意図しないブランチの変更や汚染を引き起こします。
- ワークツリー外（メインワークスペース）での作業が必要な場合（例：ワークツリーの削除）を除き、原則として常に `dir_path` を明示してください。
